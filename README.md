# Setup instructions

## Windows 
Install python >= 3.6. Then open the powershell and execute installation script:
```
.\StartNotebook.ps1
```
### On Linux (Bash) or MacOS (Zsh)
On Linux (Bash) or MacOS (Zsh):
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

### Open the notebook
To open the Jupyter notebook, run:
```
jupyter notebook
```
